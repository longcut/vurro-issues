# Vurro – Issues & Feedback

This repository is the public issue tracker for **[Vurro](https://vurro.net)**, a browser extension that enhances Substack Chat.

> The extension source code lives in a private repository. Use this repo to submit bug reports, feature requests, and questions.

---

| Browser | Link |

## How to Submit an Issue

1. Go to the [**Issues tab**](../../issues) of this repository.
2. Click **New Issue**.
3. Choose the template that best fits your need:

| Template | When to use |
|----------|-------------|
| 🐛 **Bug Report** | Something isn't working as expected |
| ✨ **Feature Request** | You'd like to see a new feature or improvement |
| ❓ **Question** | You have a question about the extension |

4. Fill in the template with as much detail as possible and submit.

---

## Tips for a Good Bug Report

- Include your **browser name and version** (e.g., Chrome 124, Firefox 126).
- Describe the **steps to reproduce** the problem.
- Describe what you **expected** to happen vs. what **actually** happened.
- Attach **screenshots or screen recordings** if possible.
- Note whether the issue happens on a specific Substack or all of them.

---

## Tips for a Good Feature Request

- Explain the **problem** you're trying to solve, not just the solution.
- Describe how you currently work around the limitation (if at all).
- Share any examples or references that illustrate the idea.

---

## Before Submitting

Please [search existing issues](../../issues) to see if your bug or request has already been reported. If it has, add a 👍 reaction or a comment with additional context rather than opening a duplicate.
